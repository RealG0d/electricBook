16. Каким образом конфигурируются URL-адреса в Django?

  • а. Путем редактирования файла конфигурации settings.py, где указываются все доступные URL-адреса проекта.
  • б. Автоматически при создании нового приложения Django, без необходимости дополнительной конфигурации URL-адресации.
  • в. С помощью команды python manage.py urls, которая генерирует и выводит список всех доступных URL-адресов в проекте.
  • г. Через файл urls.py в каждом приложении, где определяются маршруты и их обработчики представлений.
Правильный ответ: г

17. Какие способы конфигурации URL-адресов поддерживаются в Django?

  • а. Через файл urls.py в каждом приложении, где определяются маршруты и их обработчики представлений.
  • б. Через централизованный файл urls.py в корневой директории проекта, где объединяются маршруты из различных приложений.
  • в. Через классы-контроллеры, которые автоматически обрабатывают URL-запросы без необходимости явного указания.
  • г. С помощью панели администратора Django, где можно визуально настраивать и управлять всеми URL-адресами проекта.
  • д. Через файл views.py, где указываются не только функции представлений, но и URL-адреса для них.
Правильный ответ: а, б
