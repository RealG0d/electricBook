1. Установите Python: Если у вас еще не установлен Python, загрузите и установите его с официального сайта Python (https://www.python.org/downloads/).

2. Откройте любой IDE редактор кода. Например VScode или PyCharm.

3. Установите Django: Откройте терминал или командную строку и выполните команду pip install django для установки Django.

4. Проверьте установку Django: После установки выполните команду django-admin --version, чтобы убедиться, что Django успешно установлен и вы видите версию.

5. Создайте новый каталог проекта: Создайте новую папку для вашего проекта Django.

6. Перейдите в созданный каталог: Откройте терминал или командную строку, перейдите в созданную вами папку с помощью команды cd path/to/your/project.

7. Создайте виртуальную среду: Внутри вашего проекта создайте виртуальную среду с помощью команды python -m venv myenv.

8. Активируйте виртуальную среду: Активируйте только что созданную виртуальную среду. На Windows используйте myenv\Scripts\activate, а на Unix/Linux - source myenv/bin/activate.

9. Установите Django в виртуальной среде: Выполните команду pip install django в активированной виртуальной среде.

10. Создайте новый проект Django: В активированной виртуальной среде выполните команду django-admin startproject mysite, где mysite - это название вашего проекта. Пример структуры после создания проекта:

11. Проверьте запуск проекта: Перейдите в каталог вашего нового проекта (cd myproject) и выполните команду python manage.py runserver.

12. Откройте браузер и перейдите по адресу http://127.0.0.1:8000/ чтобы убедиться, что ваш проект запустился успешно.
